# SOP Akun Quotes X — Indeks Utama

> Indeks utama seluruh dokumen SOP pengelolaan Akun_Quotes di Platform_X. Dokumen dikelompokkan menjadi tiga domain sejajar: **operasional konten**, **operasional bisnis**, dan **legal & kepatuhan**.

## Struktur

```
sop/
├─ README.md                       # indeks utama (dokumen ini)
├─ operasional-konten/             # SOP konten & akun (dokumen master + lampiran + brand)
│  ├─ README.md                    # Indeks folder konten
│  ├─ SOP-Akun-Quotes-X.md         # Dokumen master (Bagian 0–11)
│  ├─ lampiran-template.md         # Template postingan, klarifikasi, log pembelajaran
│  ├─ lampiran-checklist.md        # Checklist pra-posting, moderasi, keamanan
│  ├─ lampiran-contoh.md           # Contoh format postingan, thread, & balasan
│  ├─ lampiran-monetisasi-x.md     # Peraturan boleh & tidak boleh monetisasi (acuan Content Monetization Standards X)
│  └─ panduan-brand-identitas.md   # Identitas verbal & visual, konsistensi brand (SOP-BR.1–BR.5)
├─ operasional-bisnis/             # Ketentuan bisnis & finansial antara Founder dan Co-Founder
│  ├─ README.md                    # Indeks folder bisnis
│  ├─ modal-dan-bagi-hasil.md      # Modal awal, dana cadangan, sistem bagi hasil (SOP-B.1–B.5)
│  └─ ketentuan-bisnis.md          # Kepemilikan, peran, pengeluaran, pencatatan, tata kelola (SOP-B.6–B.10)
└─ legal-kepatuhan/                # Kebijakan legal, kepatuhan, & penggunaan AI
   ├─ README.md                    # Indeks folder legal & kepatuhan
   ├─ kebijakan-hak-cipta-privasi.md # Hak cipta/fair use, takedown, data pribadi, disclosure, disclaimer (SOP-L.1–L.5)
   └─ kebijakan-penggunaan-ai.md     # Ruang lingkup & batasan AI, verifikasi, keaslian (SOP-AI.1–AI.5)
```

## Domain

- **[`operasional-konten/`](operasional-konten/README.md)** — pedoman operasional harian konten dan akun: kriteria konten & Quotes, verifikasi sumber & atribusi, styleguide, following, interaksi/komentar, moderasi, eskalasi krisis, jadwal & kalender konten, metrik & kelayakan monetisasi, dan keamanan akun. Dokumen master [`SOP-Akun-Quotes-X.md`](operasional-konten/SOP-Akun-Quotes-X.md) adalah sumber utama; lampiran memuat template, checklist, contoh, aturan monetisasi, dan panduan brand.
- **[`operasional-bisnis/`](operasional-bisnis/README.md)** — kesepakatan bisnis/finansial antara Founder dan Co-Founder: modal awal, dana cadangan langganan, sistem bagi hasil (waterfall), kepemilikan, peran, pengeluaran, pencatatan keuangan, dan tata kelola.
- **[`legal-kepatuhan/`](legal-kepatuhan/README.md)** — kebijakan legal & kepatuhan: hak cipta & fair use, penanganan klaim/takedown, perlindungan data pribadi, keterbukaan iklan, disclaimer, serta kebijakan penggunaan AI (keakuratan & keaslian).
